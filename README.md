# Starters

A set of starter projects with proper (manual) dependency injection and minimal config.

## GraphQL

GraphQL server implementations with different runtimes and deployment targets.

- [NodeJS server with TypeScript](graphql/nodejs-typescript/README.md) - Full-featured GraphQL server using Node.js and TypeScript
- [NodeJS server running in AWS Lambda](graphql/aws-lambda-nodejs/README.md) - GraphQL server optimized for AWS Lambda serverless deployment

## Front End

Front-end projects demonstrating various React and TypeScript setups with different bundling approaches.

- [TypeScript with DOM support](front-end/typescript-dom/README.md) - Pure TypeScript setup with DOM manipulation and module support
- [React without TypeScript and no bundler](front-end/plain-react/README.md) - Lightweight React setup without TypeScript or build tools
- [React with TypeScript and no bundler](front-end/react-typescript/README.md) - React with type safety but no bundler overhead

## TypeScript

TypeScript runtime setups and best practices.

- [Node TypeScript Setup](typescript/node-typescript-setup/README.md) - Starter template for Node.js TypeScript projects

## HTTPS

Secure server implementations using HTTPS.

- [Node HTTPS Server](https/nodejs/README.md) - Node.js HTTPS server with proper SSL/TLS configuration

## Kafka

Message streaming and event processing infrastructure.

- [Kafka in Docker](kafka/README.md) - Complete Kafka setup using Docker Compose for local development and testing

## Browser Testing

Automated browser and end-to-end testing frameworks.

- [Playwright Tests](browser-testing/playwright/README.md) - E2E test suite using Playwright with multi-browser support

## Benchmarking

Performance analysis and testing guides.

- [Simulating Disk Latency](benchmarking/simulatingDiskLatency.md) - Guide for setting up simulated disk latency using loopback devices and dm-delay

## Rust

Rust language examples and development patterns.

- [Modules and Unit Tests](rust/modules_unit_tests/README.md) - Examples of traits, module organization, and unit testing patterns in Rust
- [VS Code Debugging](rust/vscode_debugging/README.md) - Setup guide for debugging Rust applications in VS Code with rust-analyzer and CodeLLDB
